## Sample01
### Proposal: Hello World sample
#### Description: App to show a Hello World text in a window
*General instructions*

To restore the packages:

> **npm install restore**

To execute the app:

> **npm start**
or
> **npx electron .**


To BUILD the app:
> **npm install --save-dev @electron-forge/cli**
> **npm run make**
